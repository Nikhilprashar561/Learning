I have created **5 separate folders** for focused learning:

  Practice TypeScript concepts and examples.

- **JavaScript**

- **Nextjs**  
  Learning and experimenting with Next.js features.

- **Frontend**  
  UI components, layouts, styling, and frontend concepts.

- **Backend**  
  Backend logic, APIs, and server-side concepts.
